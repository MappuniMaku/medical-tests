# Биология — документация по добавлению вопросов

## Источник

PDF: `Lech_delo_Itogovoe_testirovanie.pdf` (79 страниц, 4.3 МБ)  
Специальность: 31.05.01 «Лечебное дело»

Структура PDF:
- Страницы 1–37: обычные тестовые вопросы (вопросы 1–250+)
- Страница 38+: **СИТУАЦИОННЫЕ ЗАДАЧИ**

## Подход к парсингу

PDF читался визуально постранично (батчами по 8–16 страниц). Правильные ответы определялись по **красной подсветке** текста в PDF. Синие/зелёные галочки — аннотации студента, они игнорировались.

В нескольких случаях PDF-метка противоречила биологии — применялось биологическое знание:

| Вопрос | PDF показывает | Использовано | Причина |
|--------|---------------|--------------|---------|
| Тест Q51 | Г (партеногенетическое) | Б (сингамное) | Определение пола при слиянии гамет — сингамное |
| Тест Q68 | Г (фенилкетонурия) | В (мукополисахаридоз) | Описание соответствует мукополисахаридозу |
| СЗ Q4 | Г (шесть пар) | Б (три пары) | После митоза клетки имеют тот же набор хромосом, что и материнская |
| СЗ Q18 | Г (♂ A₁A₁A₂A₂A₃A₃) | В (♂ A₁a₁A₂a₂A₃a₃) | Гомозиготный доминантный отец даёт всем детям рост 165 см, но у детей разный рост |
| СЗ Q45 col3 | В (свинину) | Б (говядину) | Taeniarhynchus saginatus — бычий цепень, заражение через говядину |
| СЗ Q50 col2 | Б (альвеококкоз) | А (эхинококкоз) | Дифференциальная диагностика альвеококкоза проводится с эхинококкозом, не с самим собой |

### Табличные вопросы (начиная с PDF Q40)

Некоторые ситуационные задачи имеют формат таблицы из 3 столбцов: паразит / заболевание / инвазионная стадия. Каждый такой вопрос разбивается на **3 отдельных вопроса** с последовательной нумерацией. Каждый подвопрос сохраняет полный текст условия задачи, но заканчивается одним из:

- «Назовите паразита:»
- «Назовите заболевание:»
- «Назовите инвазионную стадию:»

Варианты ответа берутся из соответствующего столбца таблицы PDF (обычно 5 вариантов а–д).

Пример: PDF Q40 → вопросы 40, 41, 42 в данных.

### Многострочные варианты ответа

Q20 содержит схемы скрещивания в вариантах. В `option.value` используются `\n`, в компоненте — `white-space: pre-line`:

```tsx
label={<span style={{ whiteSpace: 'pre-line' }}>{option.value}</span>}
```

Пример:
```typescript
{
  id: 'б',
  value: 'X^A – ген нормы, X^a – ген бесплодия\nP: ♀ X^A X^a  ×  ♂ X^A Y\nГ:   X^A, X^a       X^A, Y\nF₁: X^A X^a, X^A X^A, X^A Y, X^a Y',
}
```
